# E-Commerce Sales Performance Dashboard

## Project Overview
This project analyzes e-commerce sales data to identify key business insights related to sales performance across regions, categories, and time periods. The goal is to support data-driven decision-making using structured analysis and visualization.

## Tools & Technologies
- Excel (Pivot Tables, data cleaning)
- SQL (data aggregation and analysis)
- Power BI (interactive dashboard creation)

## Key Analysis Performed
- Sales performance by region
- Category-wise sales analysis
- Monthly sales trend analysis
- KPI tracking including Total Sales and Average Sales per Order

## Dashboard Features
- Interactive Power BI dashboard
- KPI cards for executive overview
- Region and category slicers for dynamic filtering
- Trend analysis for sales performance over time

## Business Insights
- West region contributes the highest sales, while South underperforms
- Technology category generates the maximum revenue
- Sales show seasonal variations across months

## Outcome
The dashboard provides a clear and interactive view of sales performance, enabling stakeholders to identify high-performing regions and optimize business strategies.
